# Bringo
Blog Platform

### Development

#### Requirements:
1. JDK 11
2. Scala 2.13
3. PostgreSQL 11
    * This is not required on dev machine, but you'll need to connect to a postgresql database to run the app.
4. NodeJS 10+

* IntelliJ Idea project files included to make getting started easier.

#### Build

```bash
gradlew compile
```

#### Run

```bash
gradlew run
```
